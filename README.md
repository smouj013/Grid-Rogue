# Grid Runner — PWA (v0.0.5)

Juego móvil PWA tipo runner en cuadrícula (8x24).

## Cambios v0.0.5
- Sistema de sprites configurable para:
  - player
  - bloque / coin / gem / trap / bonus
- Toggle en Opciones: Sprites ON/OFF
- Fallback automático a colores si falta algún sprite
